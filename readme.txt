一共有四个python文件：
    - monkey_decision.py: 随机生成答案
    - decision_tree.py: 定义主要的通用辅助函数（实际上决策树在此处没有定义）
    - main.py: 主函数入口，不带参数可以直接执行
    - multibrute.py: 多进程版本的主函数，搜索可能产生的最好的模型

注意：该代码只支持python2。

---

用法1：通过主函数预测并记录一个提交文件：

python main.py

修改相应的注释以实现其他功能，例如：去掉4-6行的注释能查看数据清洗前后的特征分布直方图。

用法2：暴力搜索最好的决策树模型：

python multibrute.py

修改第25行的常数，可以搜索准确率达到一定阈值的决策树模型，依赖于决策树生成时的随机性。
